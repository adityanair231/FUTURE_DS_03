# Bank Marketing Campaign Funnel Analysis

## Overview
This project performs an end-to-end exploratory data analysis (EDA) and marketing funnel analysis on the **Bank Marketing Dataset** using Python. The goal is to identify customer engagement patterns, campaign performance trends, conversion behavior, and actionable business insights for improving subscription success rates.

The analysis includes:
- Data inspection and preprocessing
- Funnel conversion analysis
- Campaign performance visualization
- Demographic trend analysis
- Correlation analysis
- Strategic business recommendations

---

## Dataset Information

The dataset used is:

- **File Name:** `bank-full.csv`
- **Separator:** `;`
- **Source:** UCI Bank Marketing Dataset

The dataset contains direct marketing campaign data from a Portuguese banking institution.

### Key Features
| Column | Description |
|---|---|
| age | Customer age |
| job | Type of occupation |
| marital | Marital status |
| education | Education level |
| balance | Average yearly account balance |
| housing | Housing loan status |
| loan | Personal loan status |
| contact | Communication type |
| duration | Last contact duration |
| campaign | Number of contacts during campaign |
| month | Last contact month |
| poutcome | Previous campaign outcome |
| y | Subscription result (yes/no) |

---

# Project Objectives

The primary objectives of this analysis are:

- Analyze customer conversion funnel
- Identify lead qualification behavior
- Measure campaign drop-off rates
- Compare communication channel performance
- Detect seasonal conversion trends
- Understand customer demographic distributions
- Generate business recommendations for improving campaign efficiency
